# Secure Login OAuth Web App

A structured PHP web application that implements secure user authentication, registration, and protected page access using an MVC-style architecture.

## Project Overview
This project demonstrates the development of a secure login system with organized folders for controllers, models, services, views, and assets. It is designed to show practical web development skills, authentication flow, and clean project structure.

## Features
- User registration system
- Secure login authentication
- Protected home page access
- MVC-based folder structure
- Reusable views and partials
- Organized CSS assets
- Backend service layer

## Project Structure
- `controller/` – Handles application logic
- `model/` – Data models
- `service/` – Business logic and authentication services
- `view/` – UI pages and partial templates
- `assets/css/` – Styling files

## Technologies Used
- PHP
- HTML
- CSS
- MVC Architecture
- GitHub

## Objective
To build a secure and maintainable web application that demonstrates authentication, code organization, and backend development skills.
